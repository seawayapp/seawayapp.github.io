Step-by-Step Solution

1

Set up coordinates

Place A at the origin. AD = AE + ED = 3 + 1 = 4 cm, so side = 4 cm.

A=(0,0) B=(0,4) C=(4,4) D=(4,0)
E=(3,0) ← AE=3cm along AD
2

Rotate E around A by 90°

Rotate E=(3,0) anticlockwise 90° around A.
This maps E to E′=(0,3) on side AB.

Rotation preserves distance, so for any P on AC:

PE = PE′
3

Apply the straight-line principle

PE + PD = PE′ + PD ≥ E′D

Minimum when P lies on the straight line from E′ to D. Minimum value = E′D.

4

Calculate E′D — 3-4-5 triangle!

E′=(0,3), D=(4,0)

E′D = √(4² + 3²) = √(16+9) = √25 = 5 cm
5

Verify P lies on diagonal AC

Point on E′D: (4t, 3−3t). Set x=y:
4t = 3−3t → t = 3/7
P = (12/7, 12/7) ✓ inside square
Minimum value of PE + PD
5 cm
